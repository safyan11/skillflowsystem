# LMS (Learning Management System) – Complete Requirement Document (SRS)

## 1. Project Overview
**Project Name:** Learning Management System (LMS)  
**Type:** Web-Based Application  
**Purpose:** Students, teachers aur admin ke liye ek digital learning platform banana jahan courses, lectures, assignments aur results manage ho saken.

## 2. User Roles
- Admin
- Teacher
- Student

## 3. Authentication System
- Login (Email/Username + Password)
- Register (Student/Teacher)
- Forgot Password
- Role-based login
- Logout

## 4. Admin Panel Features
- Dashboard (Students, Teachers, Courses stats)
- User Management (Add/Edit/Delete users)
- Course Management
- Category Management
- Reports
- System Settings

## 5. Teacher Panel Features
- Dashboard
- Course Management
- Add Lectures (Video/PDF/Text)
- Assignments (Create, Mark, Deadline)
- Quiz System
- Attendance
- Messaging

## 6. Student Panel Features
- Dashboard
- Course Access
- Watch Lectures
- Download Material
- Submit Assignments
- Attempt Quizzes
- Profile Management
- Certificates

## 7. Course System
- Title
- Description
- Category
- Teacher
- Lectures
- Assignments
- Quizzes

## 8. Progress Tracking
- Course progress bar
- Completed lectures
- Assignment scores
- Completion status

## 9. Notification System
- Assignment alerts
- New lectures
- Quiz notifications
- Admin announcements

## 10. File Management
- Upload PDFs
- Upload Videos
- Download Materials

## 11. Database Tables
- users
- courses
- lectures
- assignments
- submissions
- quizzes
- results

## 12. Technology Stack
- Frontend: HTML/CSS/JS or React
- Backend: PHP / Node.js / Python
- Database: MySQL / MongoDB

## 13. Security
- Password encryption
- Role-based access
- Secure uploads

## 14. Optional Features
- Live classes integration
- Chat system
- AI recommendations
- Payment system
